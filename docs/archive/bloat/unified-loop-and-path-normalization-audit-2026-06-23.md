## 1. Выявленные и успешно устранённые баги (Audited & Fixed Bugs)

### 1.1 Баг в нормализаторе путей (`normalizeRelativePath` в `server/workspace.js`)
* **Симптомы:** Модели Z.ai (например, `glm-4.6v-flash`) при старте чата под учётной записью `robesthud@gmail.com` передавали в инструмент `list_files` абсолютный путь контейнера `/workspace/chats/[chatId]`. Инструмент возвращал ошибку: `Path not found: chats/[chatId]`.
* **Причина:** Из-за неверной последовательности условий в `normalizeRelativePath` проверка на префикс `/workspace/` срабатывала *раньше*, чем проверка на точное равенство пути scoped-директории (`containerRoot`). В результате путь некорректно укорачивался до `chats/[chatId]`, что приводило к повторному дублированию вложенности папок при вызове `safePath` и ошибке файловой системы.
* **Решение:** Я полностью перестроил приоритет логических блоков, поместив строгое сравнение `raw === containerRoot` на первое место перед любыми срезами строк:
  ```javascript
  const containerRoot = getContainerWorkspaceRoot()
  if (raw === containerRoot || raw === '/workspace') {
    raw = ''
  } else if (raw.startsWith(containerRoot + '/')) {
    ...
  ```
  Это позволило ИИ-агенту беспрепятственно сканировать и читать рабочее пространство в любых вложенных и гостевых сессиях.

* **Решение:** 
  1. Добавил в `smartRouter.js` предварительное очищение знаков препинания на конце (`?`, `!`, `.`, `,` и пробелов) перед прохождением регулярных выражений. Простые фразы теперь классифицируются мгновенно и бесплатно за 0 мс на хосте.
  2. Внедрил гибкое отключение ИИ-классификатора в `agentLoop.js`. Теперь по умолчанию активен **Unified Runtime Mode** (как в Arena.ai), а ИИ-классификация запускается только при явной установке переменной `BROWSERAI_AI_CLASSIFICATION=1` в окружении. Сообщения идут напрямую в цикл агента, исключая любые задержки.

## 2. Новые архитектурные возможности (New Capabilities)

### 2.1 Поддержка семейства GLM (Z.ai) в двухмодельном роутере
**Файл:** `server/modelKnowledge.js`

Для полной совместимости с вашим официальным ключом Z.ai в систему маппинга родственных моделей внедрены правила для семейства GLM:
* **Апгрейд до сильной модели (`suggestStrongSibling`):** Если агент запущен на бесплатной и быстрой модели `glm-4.7-flash` или зрительной `glm-4.6v-flash`, для тяжелой умственной работы (код-ревью, планирование) рантайм автоматически переключит шаг на флагманскую супер-модель **`glm-5.2`** (с контекстом в 1 млн токенов!).
* **Даунгрейд до легкой модели (`suggestCheapSibling`):** На шагах рутинной работы с файлами система автоматически переключится на бесплатную `glm-4.7-flash`, экономя ваш баланс.
